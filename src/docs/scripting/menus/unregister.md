# UnregisterMenu
Unregistered a menu by the specified id.
> [!info]
> It can unregister only the menus which were created from the inside of the plugin. 

::: tabs
@tab Lua
```lua
--- @param id string
--- @return nil
menus:Unregister(id)
```

:::

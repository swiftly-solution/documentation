# Constructor
Generates an event object to trigger a game event from Counter Strike: 2.
::: tabs
@tab Lua
```lua
--- @param event_name string
--- @return Event
Event(event_name)
```
The event names are being listed inside [Game Events](https://swiftlycs2.net/docs/scripting/events/gameevents/).
:::

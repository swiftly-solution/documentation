# RemoveOffsetByName
Removes an offset to the pointer which is being hold by the name provided from `offsets.json`.
::: tabs
@tab Lua
```lua
--- @param offsetName string
--- @return nil
memory:RemoveOffsetByName(offsetName)
```

:::
